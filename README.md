ما هو برنامج مترجمي؟

برنامج مترجمي مصمم لترجمة الكتب القديمة المصوّرة باستخدام الذكاء الاصطناعي.
يقوم بثلاث خطوات رئيسية:

تحويل صفحات PDF إلى صور.

استخراج النصوص من الصور (OCR).

ترجمة النصوص بدقة عالية للحصول على أفضل نتائج.

كيفية استخدام البرنامج

اضغط على "اختر الملف" وحدد الكتاب PDF.

اختر لغة الكتاب الأصلية.

اختر نوع الكتاب: علمي أو أدبي.

اختر المترجم: Google، OpenAI، أو DeepSeek.

اضغط تشغيل، وسيتم حفظ الملف المترجَم في مجلد output.

شرح الخيارات

نوع الكتاب:

علمي: يحتوي معادلات ورموز، البرنامج يتجاهل المعادلات لترجمة دقيقة.

أدبي: يترجم كل النصوص بدون استثناء.

المترجمون:

Google → مجاني، مناسب للكتب الأدبية.

OpenAI → مدفوع، مناسب للكتب العلمية لدقة أعلى.

DeepSeek → مدفوع، يعتمد على API.

جودة الصورة (DPI):

DPI = عدد النقاط لكل بوصة، كلما زاد كلما كانت الصورة أوضح للنصوص.

الكتب الجديدة → 300 DPI كافية.

الكتب القديمة → 600 DPI أو أكثر.

رفع DPI يزيد حجم الملف، يمكن ضبط جودة الحفظ لتقليل الحجم.

ملفات اللغات:

بسبب حجمها الكبير، تحتاج تنزيل ملفات اللغة من GitHub ووضعها في:

tesseract/tessdata


حساسية الجملة:

لمنع ظهور نصوص غير مفهومة أو مشوهة.

مثال: ضبط الحساسية على 30% يحذف أي جملة تحتوي على أكثر من 30% رموز غير مفهومة.
